Task 3: Web Application Security
Objective
The objective of this task was to identify, understand, and demonstrate common web application vulnerabilities based on OWASP Top 10 in a controlled lab environment using DVWA and security testing tools.

Tools Used
DVWA (Damn Vulnerable Web Application)
Burp Suite
Kali Linux
Firefox Browser
Apache Web Server
SecurityHeaders.com
Activities Performed
1. SQL Injection (SQLi)
Installed and configured DVWA.
Identified SQL Injection vulnerabilities.
Extracted user information through SQL queries in a controlled environment.
Studied mitigation techniques using Prepared Statements and Parameterized Queries.
2. Cross-Site Scripting (XSS)
Stored XSS
Demonstrated persistent script injection in DVWA.
Analyzed impact on users and sessions.
Reflected XSS
Tested user input reflection through URL parameters.
Studied prevention techniques such as input validation and output encoding.
3. Cross-Site Request Forgery (CSRF)
Demonstrated CSRF attacks in a controlled lab.
Understood how unauthorized actions can be performed on behalf of users.
Implemented token-based protection mechanisms.
4. File Inclusion Attacks
Local File Inclusion (LFI)
Studied techniques used to access local files.
Analyzed potential risks and impacts.
Remote File Inclusion (RFI)
Understood remote code execution risks.
Reviewed mitigation strategies.
5. Burp Suite Advanced Testing
Intercepted HTTP requests and responses.
Modified request parameters for testing.
Performed fuzzing using Burp Suite Intruder.
Analyzed application behavior against unexpected inputs.
6. Web Security Headers
Analyzed security headers using SecurityHeaders.com.
Studied important HTTP security headers:
Content-Security-Policy (CSP)
X-Frame-Options
X-Content-Type-Options
Strict-Transport-Security (HSTS)
Configured and verified security improvements.
Skills Gained
Web Application Security Testing
OWASP Top 10 Awareness
SQL Injection Testing
XSS Analysis
CSRF Testing
Burp Suite Usage
Security Header Analysis
Vulnerability Assessment
Learning Outcomes
This task provided practical experience in identifying and understanding common web application vulnerabilities. I learned how attackers exploit insecure applications and how proper security controls can mitigate these risks.
